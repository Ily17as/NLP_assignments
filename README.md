# NLP_assignments
This repository will contain the notebooks and python code for the assignments of 3rd year NLP course at Innopolis University

# Assignments 1
In this assignment, I analyzed word frequency coverage using the Brown corpus and implemented a Byte-Pair Encoding (BPE) tokenizer from scratch. I trained the tokenizer, applied it to the corpus, and evaluated its behavior using fertility and tokenized sentence length.

How to run the notebook

Requirements:
- Python 3.x
- Packages: nltk, numpy, matplotlib
